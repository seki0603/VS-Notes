# ステータスコードの分類
* 100番台:情報レスポンス
* 200番台:成功レスポンス
* 300番台:リダイレクト
* 400番台:クライアントエラー
* 500番台:サーバエラー

## 100番台:情報レスポンス
情報処理が継続していることを示す

### 100 Continue
サーバがリクエストの受信を拒否していない

### 101 Switching Protocols
サーバがプロトコルの切り替えを要求している

## 200番台:成功レスポンス
### 200 OK
* GET...ボディにリソースが含まれる
* PUT,POST...ボディに処理結果が含まれる

### 201 Created
新規登録・画像アップロード・DBテーブル追加などに使用

### 202 Accepted
リクエストは受理されたがまだ処理が完了していないことを示す

### 204 No Content
レスポンスボディが空の時に返る

### Reset Content
ユーザーエージェントの画面をリセットするときに返る

## 300番台:リダイレクト
### 300 Multiple Choices
リクエストに対して複数のレスポンスがあることを示す

### 301 Moved Permanently
リクエストされたURLが恒久的に変更されたことを示す

### 302 Found
リクエストされたURLが一時的に変更されたことを示す

### 303 See Other
リクエストされたリソースを別のURIで取得できることを示す

### 307 Temporary Redirect
一時的リダイレクトが行われることを示す

### 308 Permanent Redirect
恒久的リダイレクトを示す

## 400番台:クライアントエラー
### 400 Bad Request
リクエストが不正
* 定義されていないメソッドを使っている
* パラメータが間違っている
...etc

### 401 Unauthorized
認証エラー
* ログイン情報が間違っている
* トークンが失効・破損している
...etc

### 403 Forbidden
認可エラー（アクセス権限がない等）

### 404 Not Found
リクエストされたリソースが存在しないことを示す

### 405 Method Not Allowed
エンドポイントは存在するが許可されていないHTTPメソッドであることを示す

### 408 Request Timeout
指定時間内にリクエストが完了しなかったことを示す

### 409 Conflict
リソースの競合が発生している

### 410 Gone
リクエストされたコンテンツがサーバから永久に削除され、転送先アドレスが無い場合を示す

### 413 Payload Too Large
リクエストヘッダ、ボディがサーバで定めている上限を超えている

### 414 URI Too Long
クライアントがリクエストしたURIが、サーバで扱える長さを超えている

### 415 Unsupported Media Type
リクエストされたデータのメディア形式にサーバが対応していない

### 429 TooMany Requests
アクセス回数が許容範囲の限界を超えている

## 500番台:サーバエラー
### 500 Internal Server Error
サーバ側に異常が発生し、正常なレスポンスが返せないことを示す
* エラーメッセージが返ることが多い
* クライアント側では解決不能
* エラーログで原因が分かる可能性がある

### 502 Bad Gateway
ゲートウェイまたはプロキシに問題が発生している

### 503 Service Unavailable
サーバがリクエストを処理する準備ができていない

### 504 Gateway Timeout
ゲートウェイとして動作するサーバが指定時間内にレスポンスを得られなかったことを示す

### HTTP Version Not Supported
リクエストしたHTTPプロトコルのバージョンにサーバが対応していないことを示す
