# Web とは

Web 　　（World Wide Web）  
Web とは、Web サイトを公開・閲覧するためのシステム  
<br>

- ハイパーテキスト＝複数の文書をハイパーリンクで相互に結びつける仕組み
- Web ＝大規模なハイパーテキスト

<br>

| 用語         | 説明                                                          |
| ------------ | ------------------------------------------------------------- |
| URL          | インターネット上の Web ページのファイルの位置や情報を示すもの |
| IP アドレス  | Web 上のコンピュータに割り当てられた数字で表す住所            |
| ドメイン     | IP アドレスに割り当てた住所の別名                             |
| サーバ       | サービスを提供するコンピュータ                                |
| DNS サーバ   | ドメイン名を IP アドレスに変換する仕組みを提供するサーバー    |
| クライアント | サーバにアクセスを要求する Web ブラウザなどを指す             |

IP アドレスはセキュリティ上、接続するたびに変わる。  
<br>

# Web の概要

## Web ページが表示されるまでの流れ

1. ブラウザ（クライアント）から URL にアクセスする
2. ブラウザは DNS サーバと通信し、URL に割り当てられた IP アドレスを要求（リクエスト）
3. DNS サーバは対応する IP アドレスをブラウザに返す（レスポンス）
4. ブラウザは確認した IP アドレスのサーバにアクセスし、HTML ファイルを要求
5. ブラウザは HTML ファイルを読み込んで表示する（レスポンス）

<br>

## 特定の URL にアクセス

### URL の構造

例）youtube 　　　`https://www.youtube.com/`

- https:// => プロトコル
- //www.youtube.com/ => ドメイン

「https 通信というプロトコルでwww.youtube.comというドメインにアクセスしたい  
という要求になる  
<br>

### プロトコルとは

ネットワーク上の通信方法を定めているもの。  
通信の用途に合わせて設定する。  
<br>

| プロトコル | 説明                                               |
| ---------- | -------------------------------------------------- |
| http(s)    | Web ページを表示する際に使用するプロトコル         |
| (s)ftp     | ファイル等の転送を行う通信プロトコル               |
| ssh        | 遠隔で外部のコンピュータを操作するためのプロトコル |

より安全な通信が必要な場合は(s)がつく。  
<br>

### ドメインとは

IP アドレスと紐づけられた Web 上の住所。特定の文字列で表現されている。  
人間が理解しやすいように住所を示す仕組み  
メールアドレスの＠以降もドメイン  
<br>

## DNS サーバが IP アドレスを返す

### IP アドレスとは

「192.123.456.789」のような特定の数字で表現された Web 上の住所  
コンピュータ、PC、スマホにも個別の番号がついている  
あらゆる電子機器に IP アドレスが存在し、それらと通信することで Web が活用できる  
<br>

### DNS サーバとは

ドメイン名＝ Domain Name System の略  
設定されているドメインを IP アドレスに変換してくれるサーバ  
Web 上では IP アドレスを識別することで通信が出来る  
<br>

## ブラウザがサーバにアクセスする

### サーバとは

サービスを提供するためのコンピュータ  
日常的に活用する PC はサーバ（提供するコンピュータ）に対して  
クライアント（提供を受けるコンピュータ）と呼ばれている。  
<br>
サーバには画像や文書を保存するなどの様々な機能があるが、  
人が直接操作することはほとんどないため、画面やキーボードはない。  
基本的に作成した Web ページのプログラムファイルはサーバの中に保管される。

<br><br>

# コマンドで Web サーバを作る

1. Web ページを作成する
2. ページを作成したディレクトリでサーバを立てる
3. ブラウザからサーバにアクセスする
4. サーバが作成されたページを探してレスポンスする

<br>
サーバの立ち上げ

`$ php -S IPアドレス：ポート番号`  
ポート番号＝ IP アドレスが住所のようなものに対し、部屋番号のようなもの。  
同じコンピュータ内で複数動作する場合、どれと通信するか指定するためにある。  
IP アドレス:ポート番号＝住所:部屋番号  
<br>

# コマンドラインショートカットキー

| ショートカットキー | 用途                     |
| ------------------ | ------------------------ |
| Control + C        | 処理を強制終了する。     |
| Control + A        | 行頭に移動する           |
| Control + E        | 文末へ移動する           |
| Control + L        | 画面をクリアにする       |
| Control + U        | 行頭まで削除             |
| Control + Y        | 最後に削除した内容を戻す |
| Control + R        | インクリメンタル検索     |
