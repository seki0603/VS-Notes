# 基本文法

`$ 動詞（コマンド） 目的語（命令内容）`

<br>

## よく使うコマンド

- cd 指定したパスのディレクトリに移動
- ls 指定したディレクトリ直下のディレクトリファイルの一覧を表示する
- pwd カレントディレクトリ(現在地)の絶対パスを表示する
- mv ファイルやディレクトリを指定した場所に保存場所を変更する
- cp ファイルやディレクトリを指定した場所にコピーする
- rm 指定したパスのディレクトリを削除
- touch 新たなファイルを作成する
- mkdir 新たなディレクトリを作成する
- open 指定したパスを GUI で開く(windows は explorer.exe)
- exit コマンドラインを終了させる

<br>

## オプション

`$ コマンド オプション 命令内容`  
オプションはコマンド毎に違う場合が多いため、「コマンド名 オプション」で調べる。  
<br>

### よくあるオプション

- ls -a 全てを表示する対象とする
- ls -l ファイルの詳細を表示する
- ls -1 ファイルを一列で表示する
- rm -r 指定したパス以下のディレクトリ/ファイルをすべて消す
- cp -r 指定したパス以下のディレクトリ/ファイルをすべてコピーする

<br><br>

# ファイルとディレクトリ

## ファイル

データを記録する際、ユーザーにとって最小の単位になるもの。  
拡張子が付いている。（非表示のものもある）  
拡張子によってなんのファイルなのか認識できる
<br>

## ディレクトリ

ファイルを分類・管理の為に格納するための入れ物  
「フォルダ」＝　「ディレクトリ」  
<br>

### ディレクトリ構造

木構造（別名：ツリー構造）　　枝分けれして広がっていく構造  
<br>

### 特別なディレクトリ

- ルートディレクトリ(/)　　　 PC 上で一番上(根底)にあるディレクトリ
- ホームディレクトリ(~)　　　ユーザが自由に使えるディレクトリの一番上のディレクトリ
- カレントディレクトリ(.)　　　現在位置を示すディレクトリ

<br><br>

# パスの概念

ファイルやディレクトリの場所を示すパソコン上の住所

- 絶対パス　　　現在位置に関わらず、絶対的に表現されるパス
- 相対パス　　　現在位置から見た、相対的に表現されるパス

<br>

| カレントディレクトリ(現在位置)  | 日本(Japan)ディレクトリ                        |
| ------------------------------- | ---------------------------------------------- |
| 東京(tokyo)までの絶対パス(住所) | /Users/ユーザー名/coachtech/Japan/kantou/tokyo |
| 東京(tokyo)までの相対パス(住所) | kantou/tokyo                                   |

<br>

現在地より前のパスを指定するとき  
|コマンド|説明|
|-------|------|
|cd ../|一つ前に戻る|
|cd ../../|二つ前に戻る|
<br>

## 特殊なパス

| パス   | 説明                       |
| ------ | -------------------------- |
| ./     | カレントディレクトリを指す |
| ../    | 一つ前のディレクトリを指す |
| ../../ | 二つ前のディレクトリを指す |
| ~      | ホームディレクトリを指す   |
| /      | ルートディレクトリを指す   |
