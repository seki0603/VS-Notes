# TransitionGroupの利用
Transitionコンポーネントと違い、複数の要素にアニメーションをてきようできるため、リストの要素にアニメーションをつけるのに適している。
```
<script setup>
import { ref } from 'vue'

const fruits = ref(['Apple', 'Banana', 'Grape'])
const newFruit = ref('')
</script>

<template>
  <h1>Animation</h1>
  <input v-model="newFruit" type="text">
  <button @click="fruits.unshift(newFruit)">Add</button>
  <TransitionGroup name="fade">
    <div v-for="(fruit, index) in fruits" :key="fruit" @click="fruits.splice(index, 1)">{{ fruit }}</div>
  </TransitionGroup>
</template>
```
* 元々複数対応しているため、mode属性には対応不可
* `<TransitionGroup tag="html要素">`と記述することで、指定した要素にまとめることができる
* v-moveクラスによって、要素が追加・削除されるときの周りの要素にアニメーションを指定できる
<br>

## tag属性
Vueファイルの記述
```
<template>
  <TransitionGroup name="fade" tag="div">
    <div v-for="(fruit, index) in fruits" :key="fruit" @click="fruits.splice(index, 1)">{{ fruit }}</div>
  </TransitionGroup>
</template>
```
<br>

効果
```
<div>
  <div>fruit1</div>
  <div>fruit2</div>
  <div>fruit3</div>
</div>
```
<br>

## v-moveクラス
```
<style scoped>
.fade-enter-from {
  opacity: 0;
}
.fade-enter-active {
  transition: opacity 1s;
}
.fade-enter-to {
  opacity: 1;
}
.fade-move {
  transition: transform 1s;
}
.fade-leave-from {
  opacity: 1;
}
.fade-leave-active {
  transition: opacity 1s;
  position: absolute;
}
.fade-leave-to {
  opacity: 0;
}
</style>
```
* .v-moveで要素が追加された時の周りのアニメーションを指定する
* `.v-leave-active`に`position: absolute;`を指定することで、要素が消えるときの周りの移動が滑らかになる。
