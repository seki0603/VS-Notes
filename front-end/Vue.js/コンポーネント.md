# コンポーネントの埋め込み
```
<script setup>
import CountUp from './CountUp.vue';
</script>

<template>
  <h1>App</h1>
  <CountUp />
</template>
```

* 親コンポーネントのscript内で子コンポーネントをインポート
* templateタグ内に`<コンポーネント名 />`で埋め込み

<br><br>

# コンポーネントのグローバル登録
main.js内にインポートすることで、importなしでどのコンポーネントにも埋め込めるようにする。  

## main.js
```
import { createApp } from 'vue'
import App from './App.vue'
import BaseIcon from './BaseIcon.vue'

const app = createApp(App)
app.component('BaseIcon', BaseIcon)
app.mount('#app')
```
* グローバル登録するコンポーネントをインポート
* `app.mount`より先に`app.component('BaseIcon', BaseIcon)`を指定
<br>

## 埋め込み先のコンポーネント
```
<script setup>
</ script>

<template>
    <BaseIcon />
</ template>
```
* import不要
* templateタグ内にグローバル登録したコンポーネントを埋め込み

<br><br>

# 属性の継承
親コンポーネント
```
<script setup>
import CountUp from './components/CountUp.vue';
</script>

<template>
  <h1>App</h1>
  <CountUp id="base-button" class="border" />
</template>
```

子コンポーネント
```
<template>
  <div>
    <h2>CountUp</h2>
    <BaseIcon />
  </div>
</template>
```

* 親コンポーネントに埋め込まれているタグから属性を渡すことができる
* 子コンポーネントの外側が一つのタグでまとまっている場合、一番外側の外側のタグに属性が適用される
    * コンポーネントでも属性を指定していた場合、ディレクティブは継承されず、id・classは継承されたものと直接指定したものが両方適用される。

<br><br>

# CSSの適用範囲
どのコンポーネントのstyleタグに書いても、全てのコンポーネントに適用される。
<br>

## 適用範囲をそのコンポーネントに限定する
```
<style scoped>
.red {
  color: red;
}
</style>
```
* styleタグ内にscopedをつけることで、CSSの適用範囲をそのコンポーネント内部に限定することができる。
<br>

※コンポーネント埋め込んでいる場合、子コンポーネントの一番外側のタグにもスタイルができようされてしまう。(子コンポーネントが一つのタグでまとめられている場合)

<br><br>

# CSSのグローバル登録
viteの機能を使ってCSSファイルを全てのコンポーネントに適用する  
（index.htmlのheadタグ内に挿入される）
## ファイル作成
src/assets/ファイル名.css作成
<br>

## main.jsへのインポート
`import './assets/ファイル名.css'`
