# props で値を渡す

## 値の受け取り

子コンポーネント側

```
export function Headline(props) {
  return (

    // 中略

  );
}
```

- コンポーネント関数の引数に props（他の値でも OK）を渡す

## 値の渡し方

親コンポーネント側

```
// 略

<Headline page="about" />
```

## 受け取った値の表示

子コンポーネント側

```
{props.page}
```

- 複数の値を渡すことができる

## 色んな値の渡し方

```
<Headline
  page="about" // 文字列
  number={1111} // 数値
  array={[1, 2, 3]} // 配列
  obj={{ foo: "foo", bar: "bar" }} // オブジェクト
  boolean // 真偽値
  comp={<div>foo</div>} // コンポーネント
  onClick={() => 処理内容} // 関数
/>
```
* 真偽値はtrueの場合のみ省略可能
    * boolean={false}

### childrenで渡す
```
<Headline>
    // 値
</Headline>
```
* {props.children}で受け取ることができる
* コンポーネントを渡すときなどに使う
