# jsconfig.json
プロジェクト直下にjsconfig.jsonを作成
```
{
    "compilerOptions": {
        "baseUrl": "."
    }
}
```
* ``baseUrl": "."`により、パスの起点を.に設定

# 絶対パスインポート
相対パス指定`import { Headline } from "../Headline";`を絶対パス指定`src/components/Headline";`に変更

# エディター補完の設定
VS Codeなどのエディターによるパス補完は初期設定では相対パスで行われる

## 設定変更
* エディター上で「ctrl + ,」→設定を開く
* Workspace
    * User = 全てに適用する設定、Workspace = 今回のプロジェクトのみに適用する設定
* importで検索
* JavaScript > Preferences: Import Modules Specifierをnon-relativeに変更
